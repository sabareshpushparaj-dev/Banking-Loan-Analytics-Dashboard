# Banking-Loan-Analytics-Dashboard

This project presents an interactive Banking Loan Analysis Dashboard built in Power BI.
It provides comprehensive insights into loan exposure, customer income segmentation, credit risk profiling, and portfolio performance.

The dashboard analyzes key metrics such as total bank loans, risk weighting, income bands, credit card exposure, superannuation savings, deposits, and business lending.

Advanced DAX measures were implemented to calculate risk-adjusted loans, weighted risk averages, loan-to-income ratios, and portfolio concentration metrics.

This dashboard is designed to support strategic decision-making for risk management, lending optimization, and executive reporting.
